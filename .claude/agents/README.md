# 東方の三賢者 (The Three Magi)

Sui Auto LP システムの開発、運用、監査をサポートする3つの専用AIエージェントの定義（プロンプト）です。各エージェントはそれぞれ異なる役割と専門性を持ち、システム全体の安全性・収益性・安定性を高めるために協調します。

## 🧙‍♂️ エージェント一覧

### 🥇 [Melchior (黄金の賢者) — The Strategist](./melchior.md)
* **役割**: 戦略パラメータの分析、収益性の評価、最適化の提案
* **責務**: リバランス閾値やレンジ幅の分析、ROI計算、シミュレーションを通じた戦略提案（直接のコード変更は行わない）
* **象徴**: 黄金 (Gold) — 価値の最大化

### 🌫️ [Balthasar (乳香の賢者) — The Auditor](./balthasar.md)
* **役割**: セキュリティ監査、QA (品質保証)、コードレビュー
* **責務**: Gemini CLI を用いた外部AI視点での非依存なロジック検証、DeFi リスク分析、型安全性やエッジケースの確認
* **象徴**: 乳香 (Frankincense) — 検証の神聖さ

### 💧 [Caspar (没薬の賢者) — The Operator](./caspar.md)
* **役割**: ライブポジションの監視、ログ分析、デプロイ管理、システムヘルスの維持
* **責務**: 本番稼働中のサービス状態・ログの監視、ヘルスチェック手続きの実行、デプロイや設定変更の安全な実施
* **象徴**: 没薬 (Myrrh) — 資産の保存と保護
