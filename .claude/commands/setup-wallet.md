description: Setup or verify Sui wallet configuration
allowed-tools: Bash, Read
---

Suiウォレットのセットアップ・確認を行ってください。

1. `sui client active-env` で現在のネットワーク確認
2. `sui client active-address` でアクティブアドレス確認
3. `sui client balance` で残高確認
4. `sui client envs` で登録済み環境一覧
5. 問題があれば修正手順を提示

**注意**: 秘密鍵やmnemonicを画面に表示しないこと。
