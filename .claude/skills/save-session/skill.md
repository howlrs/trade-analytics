# save-session

セッションの要約を `docs/sessions/` に保存するスキル。

## トリガー

ユーザーが「セッションを保存」「save session」「今日の作業を記録」と指示した場合に使用。

## 手順

1. 現在の日時を `YYYYMMDD_HHMM` 形式で取得する
2. セッション中に行った作業の要約を作成
3. `docs/sessions/<日時>_<概要>.md` として保存

## セッションファイルの構成

```markdown
# セッション: <概要>

日時: YYYY-MM-DD HH:MM

## 実施した作業
- <作業1>
- <作業2>

## 得られた知見
- <知見1>
- <知見2>

## 次のアクション
- <TODO1>
- <TODO2>

## 関連ファイル
- <作成・修正したファイルパス>
```
