## 検証したシグナル (Tier-1)

| シグナル | 定義 | 全期間 Q5-Q1 |
|---|---|---|
| Vol Ratio 7d | volume / rolling_mean(168) | +13〜+43bps |
| PV Divergence | momentum_24h × -(vol_ratio_24h - 1) | -10〜-32bps |
| Vol Imbalance 8h | signed_volume.rolling_sum(8) / volume.rolling_sum(8) | -5〜+30bps |

## 結論: 実用水準に達しない

### ウォークフォワード (Train < 2025-09-01 / Test >= 2025-09-01)

- **BTC/ETH**: Trainで強いシグナル (Sharpe 2-3) → **Testで完全崩壊** (Sharpe < 0)
- **SOL/SUI**: Testで一部残存するが、流動性の低さによるインサンプル偏りの疑い
- WF比率 (Test/Train) がシグナル・通貨により不安定 → 構造的アルファとは言えない

### 複合シグナル

```
Composite = z(vol_ratio_7d) - z(pv_divergence) + z(vol_imbalance_8h)
```

- ETH/SOL/SUIでは個別より改善 (最大+46bps)
- BTC では改善なし
- **Testでは BTC -0.25, ETH -0.34 とマイナスSharpe**

### FR/OI追加効果

```
Composite+ = Composite - z(FR) + z(OI変化率4h)
```

- **全通貨で悪化** (BTC -11bps, SOL -19bps)
- OHLCVシグナルとデリバティブの情報が重複 or ノイズ増加

### シグナル減衰

- 半減期は約 8h (8h保有の設計と整合)
- lag24h で不自然な再上昇あり → 周期性アーティファクト (日次サイクル) の可能性

### エクイティカーブ特性

- 勝率 19-22% → テール依存型 (少数の大勝ちに依存)
- MaxDD が累積リターンを上回る場面あり → 実運用には耐えられない

## 教訓

1. **線形5分位分析の限界**: R² 0.001-0.008 のシグナルを線形結合しても予測力は積み上がらない
2. **大型通貨ほどアルファが消えやすい**: BTC/ETH は効率的市場に近く、OHLCV情報は即座に織り込まれる
3. **全期間の統計量は過大評価する**: Train/Testで結果が反転するため、必ずウォークフォワードで検証すべき
4. **複合スコアのz変換は万能ではない**: 異なるシグナルの加算が最適とは限らない (非線形交互作用を見落とす)

## 次に試すべきアプローチ

1. **非線形モデル** (GBM等) で特徴量間の交互作用を捕捉
2. **ボラティリティ・レジーム分離**: 高ボラ期のみ機能するシグナルがあるか
3. **デリバティブ主導**: FR極端値・OI急変をプライマリ、OHLCVはフィルタとして格下げ
4. **時間帯フィルタ**: 東京/ロンドン/NYオープン前後に限定して検証
