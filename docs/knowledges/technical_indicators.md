# テクニカル指標リファレンス

## 1. トレンド系指標

### SMA（単純移動平均）

```
SMA(n) = (P₁ + P₂ + ... + Pₙ) / n
```

| 用途 | 短期 | 中期 | 長期 |
|------|------|------|------|
| デイトレード | 5, 10 | 20, 25 | 50 |
| スイング | 20 | 50 | 100 |
| 長期 | 50 | 100 | 200 |

### EMA（指数平滑移動平均）

```
k = 2 / (n + 1)
EMAₜ = Pₜ × k + EMAₜ₋₁ × (1 - k)
```

典型パラメータ: 9, 12, 21（短期）/ 26, 50（中期）/ 100, 200（長期）

### MACD

```
MACDライン  = EMA(12) - EMA(26)
シグナルライン = EMA(MACDライン, 9)
ヒストグラム  = MACDライン - シグナルライン
```

| シグナル | 条件 | 意味 |
|---------|------|------|
| ゴールデンクロス | MACDがシグナルを下から上抜け | 買い |
| デッドクロス | MACDがシグナルを上から下抜け | 売り |
| ゼロライン上抜け | MACD > 0 | 強気確認 |
| 強気ダイバージェンス | 価格安値更新、MACD安値更新せず | 反転上昇示唆 |
| 弱気ダイバージェンス | 価格高値更新、MACD高値更新せず | 反転下落示唆 |

### ADX

```
TR = max(高値-安値, |高値-前日終値|, |安値-前日終値|)
+DI(14) = (+DM平滑 / ATR(14)) × 100
-DI(14) = (-DM平滑 / ATR(14)) × 100
DX = |+DI - -DI| / |+DI + -DI| × 100
ADX(14) = DXの14期間平滑平均
```

| ADX値 | 状態 |
|-------|------|
| 0-20 | レンジ（オシレーター系有効） |
| 20-25 | トレンド形成初期 |
| 25-40 | 明確なトレンド |
| 40-60 | 強いトレンド |
| 60+ | 極めて強い（過熱注意） |

### ボリンジャーバンド

```
MB = SMA(20)
UB = MB + 2σ
LB = MB - 2σ
Bandwidth = (UB - LB) / MB × 100
```

- スクイーズ（収縮）→ 大きな動きの前兆
- ウォーキング・ザ・バンド → 強トレンド継続

## 2. オシレーター系指標

### RSI（14期間）

```
RS = 上昇幅平均 / 下落幅平均
RSI = 100 - (100 / (1 + RS))
```

| RSI値 | 状態 |
|-------|------|
| 70+ | 買われすぎ |
| 30- | 売られすぎ |
| 50超 | 強気領域 |
| 50割 | 弱気領域 |

ダイバージェンスが特に有効。

### ストキャスティクス

```
%K = (終値 - n期間最安値) / (n期間最高値 - n期間最安値) × 100
%D = %K の m期間 SMA
```

典型パラメータ: %K(14), %D(3), スムージング(3)
- 80+: 買われすぎ / 20-: 売られすぎ

### CCI

```
TP = (高値 + 安値 + 終値) / 3
CCI = (TP - SMA_TP) / (0.015 × Mean Deviation)
```

+100超: 買われすぎ / -100以下: 売られすぎ

## 3. ボリューム系指標

### OBV
```
終値 > 前日終値 → OBV += 当日出来高
終値 < 前日終値 → OBV -= 当日出来高
```

価格とOBVのダイバージェンスがブレイクアウト予兆。

### VWAP
```
VWAP = Σ(TP × Volume) / Σ(Volume)
```

VWAP上 = 強気、VWAP下 = 弱気。機関投資家の基準値。

### Volume Profile
- **POC**: 最大出来高価格帯 → 強力なS/R
- **VAH/VAL**: 出来高70%集中ゾーンの上限/下限
- **HVN**: 出来高多 → 価格留まりやすい
- **LVN**: 出来高少 → 価格が素早く通過

## 4. 暗号資産特有の指標

### Funding Rate

```
FR > 0 → ロングがショートに支払い（買い過熱）
FR < 0 → ショートがロングに支払い（売り過熱）
```

| 状況 | 示唆 |
|------|------|
| FR極端に高い（+0.1%超/8h） | ロング過熱、下落リスク |
| FR極端に低い（-0.05%超/8h） | ショート過熱、スクイーズリスク |
| 正常範囲（±0.01-0.03%/8h） | トレンドフォロー有効 |
| 高値更新+FR低い | 持続性のある上昇 |

**アービトラージ**: 現物ロング + 先物ショートでFR受け取り（デルタニュートラル）

### Open Interest

| 価格 | OI | 解釈 |
|------|-----|------|
| 上昇 | 増加 | 強気（新規ロング参入） |
| 上昇 | 減少 | ショートカバー（持続性弱い） |
| 下落 | 増加 | 弱気（新規ショート参入） |
| 下落 | 減少 | ロング投げ売り（持続性弱い） |

OI急増 = 大きな変動の前兆。OI過去最高+高FR = スクイーズリスク大。

### Long/Short Ratio

- **Top Trader L/S Ratio**: 大口ポジション比率（重要）
- **Global L/S Ratio**: 全体（群衆心理）
- 極端なロング比率 → 逆張り売りシグナル
- アカウント数ベースと建玉金額ベースで解釈が異なる

### Liquidation

- **ロング清算**: 価格下落 → さらなる売り圧力（カスケード）
- **ショート清算**: 価格上昇 → さらなる買い圧力
- 清算ヒートマップ: 清算集中価格帯が「マグネット」として価格を引き寄せる

### Exchange Inflow/Outflow

| 動き | 解釈 |
|------|------|
| Inflow増加 | 売却準備 → 弱気 |
| Outflow増加 | 長期保有移行 → 強気 |
| 大口Inflow | 大量売却の前兆 |
| Outflow継続+価格上昇 | 需給ひっ迫 → 強気継続 |

## 5. Pythonライブラリ・計算方法

### polars式ベース（推奨）

polarsのExpression APIで直接計算する。外部ライブラリ不要。

```python
import polars as pl

df = df.with_columns(
    pl.col("close").rolling_mean(window_size=20).alias("sma_20"),
    pl.col("close").ewm_mean(span=12).alias("ema_12"),
)
```

### TA-Lib（numpy配列経由）

高速なC実装。polarsからはnumpy変換して使用。

```python
import talib
close = df["close"].to_numpy()
df = df.with_columns(pl.Series("rsi_14", talib.RSI(close, timeperiod=14)))
```

| 項目 | polars式 | TA-Lib |
|------|----------|--------|
| 速度 | 高 | 最高（C実装） |
| 指標数 | 基本指標 | 約200 |
| インストール | 不要 | 難（C依存） |
| polars統合 | ネイティブ | 要numpy変換 |
| ローソク足パターン | なし | 61種 |
| 推奨用途 | 一般的な分析 | 高速バックテスト・パターン検出 |

暗号資産独自指標（OI・FR等）はいずれも非対応 → API直接取得で自前実装。
